# Template Website Workshop & Sertifikasi

[![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)](https://developer.mozilla.org/en-US/docs/Glossary/HTML5)
[![Bootstrap](https://img.shields.io/badge/Bootstrap-563D7C?style=flat&logo=bootstrap&logoColor=white)](https://getbootstrap.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Proyek ini adalah template antarmuka (User Interface) statis untuk landing page acara workshop dan pelatihan bersertifikat. Dibangun dengan fokus pada kecepatan, desain responsif, dan kemudahan kustomisasi tanpa memerlukan konfigurasi backend yang rumit.

## 🚀 Fitur Utama

*   **100% Responsif:** Desain yang menyesuaikan dengan berbagai ukuran layar (Desktop, Tablet, dan Mobile) menggunakan sistem grid Bootstrap.
*   **Hero Section Menarik:** Bagian header utama yang dirancang untuk menyoroti judul workshop, tanggal pelaksanaan, dan tombol _Call-to-Action_ (CTA) pendaftaran.
*   **Jadwal Acara (Rundown):** Layout timeline yang rapi untuk menampilkan susunan acara workshop dari awal hingga akhir.
*   **Katalog Pembicara:** Kartu profil (Cards) untuk menampilkan foto, nama, dan latar belakang pemateri.
*   **Formulir Pendaftaran (UI):** Antarmuka formulir registrasi yang bersih dan mudah diisi oleh calon peserta.
*   **Preview Sertifikat:** Bagian khusus untuk menampilkan contoh desain sertifikat digital yang akan didapatkan oleh peserta.

## 🛠️ Tech Stack

Proyek ini menggunakan teknologi web dasar sehingga sangat ringan dan mudah dimodifikasi:
*   **HTML5:** Struktur semantik halaman web.
*   **CSS3 & Bootstrap 5:** Styling, tata letak, dan komponen antarmuka yang responsif (Navbars, Cards, Modals, Buttons).
*   *(Opsional)* **Vanilla JavaScript / Bootstrap JS:** Untuk interaktivitas dasar seperti toggle menu mobile atau modal pop-up.

## 📦 Cara Penggunaan

Karena proyek ini bersifat statis (tidak memerlukan server lokal atau instalasi *dependencies* khusus), kamu bisa langsung menjalankannya dengan langkah berikut:

1.  **Clone Repositori**
    ```bash
    git clone [https://github.com/username-kamu/nama-repo-kamu.git](https://github.com/username-kamu/nama-repo-kamu.git)
    cd nama-repo-kamu
    ```

2.  **Buka File Utama**
    Klik ganda pada file `index.html` atau buka file tersebut langsung menggunakan web browser (Chrome, Firefox, Safari, Edge).

3.  **Kustomisasi**
    Buka proyek di Text Editor favoritmu (seperti VS Code atau Sublime Text) dan mulailah mengubah teks, gambar, atau warna sesuai dengan tema workshop.
